# React Forms
This project demonstrates how to build and validate forms in React using `react-hook-form`. It covers form state management, validation rules, and dynamic rendering of submitted data.
---
## Features

- Form handling using `react-hook-form`
- Validation for required fields, minimum length, and patterns
- Real-time error messages
- Dynamic display of submitted user data
- Responsive UI using Tailwind CSS
---

## Tech Stack
- React (Vite)
- react-hook-form
- Tailwind CSS
