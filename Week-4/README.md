# ATP - Week 4

This repository contains the assignments and practice exercises completed during Week 4 of the ATP Training Program.

The focus of this week was on building strong foundations in HTML and CSS, including webpage structure, semantic elements, nested lists, and styling techniques.

---

## Key Focus Areas

- HTML page structure
- Semantic HTML elements
- Ordered and unordered lists
- Nested elements and content organization
- CSS selectors and styling
- Colors, typography, and spacing
- Frontend layout development
