## Assignments – Day 1

1. Find the bigger number between two numbers
2. Find the biggest number among three numbers
3. Calculate the sum of marks in `[90, 78, 65, 98]`
4. Find the smallest element in a marks array
5. Write a function that receives three numbers and returns the biggest number
6. Write a function that receives an array and returns the sum of all elements
7. Write a function that receives an array and a search element, and returns its index
   - Return `"not found"` if the element does not exist
