# ATP - Week 2

This repository contains the assignments, practice programs, and backend exercises completed during Week 2 of the ATP Training Program.

During this week, the focus was on improving JavaScript problem-solving skills, mastering ES6+ features, and building a strong foundation in backend development using Node.js.

---

## Key Focus Areas

- JavaScript problem-solving
- Arrays and objects manipulation
- ES6+ concepts
- Real-world mini applications
- Backend fundamentals with Node.js
- API request handling and testing
